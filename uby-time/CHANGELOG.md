# Changelog

All notable changes to `uby-time` will be documented in this file.

## 0.1.0 - 2026-06-17

### Added

- Initial minimum Python reference implementation for UBY Working Draft 0.1.0.
- Level 1 JD / ISO / astronomical year / BC year conversion helpers.
- UBY full, magnitude, scientific, academic mnemonic and friendly mnemonic formatting.
- UBY expression parser for the five supported notation forms.
- Basic validation messages.
- Optional `astropy`-based redshift conversion entry point.
- CLI entry point.
- Initial test vectors.
